Responda, para que servem os comandos:

Alert:É uma função embutida que exibe uma caixa de diálogo com uma mensagem para o usuário.Essa mensagem pode ser usada para informar o usuário sobre algo importante, como um aviso ou uma mensagem de erro.

Let:Permite que você declare variáveis limitando seu escopo no bloco, instrução, ou em uma expressão na qual ela é usada.

//:Essa tag é utilizada para deixar uma mensagem no código sem modificar ou interferir na estrutura ou na pagina

If:Significa "se" e é como usamos os operadores lógicos de uma linguagem. Ele condiciona um comando a uma condição qualquer, executando-o apenas se ela for verdadeira.

Else:Significa "senão", permite maior controle sobre o fluxo de código que o comando mais básico if, por permitir múltiplos testes serem agrupados juntos
