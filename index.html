Let:Permite que você declare variáveis limitando seu escopo no bloco, instrução, ou em uma expressão na qual ela é usada.

If:Significa "se" e é como usamos os operadores lógicos de uma linguagem. Ele condiciona um comando a uma condição qualquer, executando-o apenas se ela for verdadeira.

Else:Significa "senão", permite maior controle sobre o fluxo de código que o comando mais básico if, por permitir múltiplos testes serem agrupados juntos

Diferença entre == e === em JavaScript:O operador == verifica a igualdade de valor e realiza a conversão de tipo, se necessário. O operador === realiza uma comparação estrita, verificando tanto a igualdade de valor quanto de tipo, sem realizar conversões automáticas.

O que é um objeto em JavaScript e como você cria um?:É uma estrutura na qual você pode juntar variás variáveis e depois acessar essas variáveis de forma independente. No Jva coloque a palavra class e abra uma chave, ou seja, Class {}, e tudo que estiver dentro dessa chave será um objeto.

O que é um array em JavaScript e como você pode adicionar ou remover elementos de um array?:Um array é um tipo especial de variável utilizado para armazenar uma coleção de dados, ou seja, armazenar múltiplas informações de uma só vez. Uma das maneiras mais comuns de remover um item de um array em TypeScript é utilizando o método splice(). Esse método permite remover elementos com base no índice e também pode ser usado para substituir elementos.Para adicionar elementos a um Array, é preciso definir uma posição. Para isso, basta inserir um item no final do Array, utilizando o método push ().

Qual é o significado comando While?:Esta instrução é usada quando não sabemos quantas vezes um determinado bloco de instruções precisa ser repetido. Por exemplo, estou no mercado, se eu colocar que eu preciso de esmalte e tiver no corredor do determinado produto (será verdadeiro que eu quero um esmalte) por tanto o codigo será feito, já a partir do ponto que eu não tiver mais no corredor do esmalte, o codigo será falso e não funcionará.
